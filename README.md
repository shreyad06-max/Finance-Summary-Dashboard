# Finance-Summary-Dashboard
This project is an interactive Finance Summary Dashboard built in Microsoft Excel to simplify financial reporting and analysis. It provides a clear overview of key business metrics, including Revenue, Expenses, Profit & Loss, Maximum Revenue, and Maximum Expense.

The dashboard uses Pivot Tables, Pivot Charts, Slicers, and Excel formulas to create dynamic visualizations that allow users to explore financial data with ease. It also includes category-wise revenue and expense analysis, department-wise revenue vs. expense comparison, and monthly profit tracking to support better business decisions.

🔹 Key Highlights
Interactive KPI cards for quick financial insights
Revenue and expense breakdown by category
Department-wise revenue vs. expense comparison
Monthly profit analysis
Dynamic filtering using slicers
Clean and user-friendly dashboard design

🛠️ Tools Used
Microsoft Excel
Pivot Tables & Pivot Charts
Slicers

Data Visualization
